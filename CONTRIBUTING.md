# Участие в проекте

Спасибо, что хотите улучшить **netfix**.

## Как помочь

1. **Issues** — опишите дистрибутив, версию NetworkManager и что показывает `netfix` / лог из `/var/log/network-reset-*.log`.
2. **Pull requests** — небольшие, понятные изменения; один PR — одна задача.
3. **Документация** — правки README, примеры для вашего дистрибутива.

## Перед PR

```bash
bash -n netfix.sh
bash -n install.sh
```

Проверьте сценарий на тестовой VM, если меняете этап 1 или 2.

## Стиль кода

- Bash: `set -euo pipefail`, явные имена функций.
- Сообщения пользователю — на русском, как в текущем скрипте.
- Не ломайте обратную совместимость alias и путей `/usr/local/bin/netfix` без обсуждения в issue.

## Безопасность

Уязвимости и риски сценариев (SSH, продакшен) — через **Issues** с пометкой `security`, без публичных эксплойтов.
